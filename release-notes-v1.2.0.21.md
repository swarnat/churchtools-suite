# Release Notes v1.2.0.21

## Zusammenfassung
Dieses Release verbessert die UX der Hero-View und behebt den Fehler beim Klick auf das Modal-Bild.

## Enthaltene Änderungen

### Modal / Bildklick
- Klick auf das Modal-Bild springt nicht mehr zur Homepage, wenn kein gültiger Bild-Link vorhanden ist.
- Bild-URL-Auflösung im Frontend ist robuster und toleriert unterschiedliche Payload-Varianten.

### Hero UX
- Neue Presets für Bild/Höhe:
  - Kompakt
  - Standard
  - Hero
- Neue Option für Mobile-Optimierung der Hero-View.
- Optimierte Default-Werte für Höhe, Abstände und Titel-Darstellung.
