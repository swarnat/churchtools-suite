# Release Notes v1.2.0.18

## Zusammenfassung
Dieses Release verbessert die Auffindbarkeit der neuen Bild-/Hero-Einstellungen in Elementor und optimiert die Bilddarstellung im Modal.

## Enthaltene Änderungen

### Elementor
- Stil-Tab gruppiert in klaren Bereichen:
  - Bild & Hero
  - Farben & Layout
- Die Optionen Bilddarstellung (Cover/Contain) und Titel-Schriftgröße (Hero) sind damit direkt unter Stil verfügbar.

### Modal / Frontend
- Modal-Bilder verwenden jetzt contain statt cover, damit das Bild vollständig sichtbar bleibt.
- Modal-Bilder sind klickbar und öffnen das Originalbild in einem neuen Tab.

## Versionen
- Core Plugin: 1.2.0.18
- Elementor Addon: 0.6.19
- Posts Sync Addon: 0.1.4
- Presentations Addon: 0.1.0

## Hinweise
- Nach dem Update ggf. Browser-Cache leeren.
- Falls Elementor noch alte Controls zeigt, Editor einmal neu laden.
