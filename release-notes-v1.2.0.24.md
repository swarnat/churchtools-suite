# Release Notes v1.2.0.24

## Zusammenfassung
Dieses Release verbessert die Carousel-UX: Navigationspfeile werden bei nur einem Termin nicht mehr angezeigt.

## Enthaltene Änderungen

### Carousel Navigation
- Prev/Next-Pfeile werden serverseitig nur noch gerendert, wenn mehr als ein Termin vorhanden ist.
- Zusätzlicher Frontend-Fallback blendet Pfeile und Pagination aus, sobald keine Navigation möglich ist (z. B. nur ein Slide oder responsive ohne Navigationsschritt).

- Posts Sync Addon: 0.1.4
