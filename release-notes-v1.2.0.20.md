# Release Notes v1.2.0.20

## Zusammenfassung
Dieses Release behebt die offenen Probleme in der Hero-View (Elementor + Rendering).

## Enthaltene Änderungen

### Elementor
- Schalter `Titel anzeigen (Hero)` wieder verfügbar (Core und Addon Widget).
- `Titel-Schriftgröße (Hero)` wird korrekt an den Shortcode übergeben.

### Rendering / Frontend
- `hero_title_font_size` ist im Carousel-Shortcode korrekt als Attribut registriert.
- Hero-Template berücksichtigt `show_title` wieder.
- Contain-Bilddarstellung im Hero robuster umgesetzt, damit das Bild vollständig sichtbar bleibt.

- Core Plugin: 1.2.0.20

- Posts Sync Addon: 0.1.4
