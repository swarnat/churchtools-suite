# Release Notes v1.2.0.19

## Zusammenfassung
Dieses Release stellt die Update-Erkennung in WordPress wieder eindeutig sicher, indem die Versionen erneut angehoben wurden.

## Enthaltene Änderungen

### Release / Updater
- Core-Version erhöht auf `1.2.0.19`.
- Elementor-Addon-Version erhöht auf `0.6.20`.
- Monorepo-Release mit konsistenten Asset-Versionen.

### Bereits enthaltene Features aus dem Vorgänger
- Elementor Stil-Tab mit klarer Gruppierung (`Bild & Hero`, `Farben & Layout`).
- Bilddarstellung (`Cover/Contain`) im Stil-Tab leichter auffindbar.
- Modal-Bilder vollständig sichtbar (`contain`) und klickbar (Originalbild in neuem Tab).

## Versionen
- Core Plugin: 1.2.0.19
- Elementor Addon: 0.6.20
- Posts Sync Addon: 0.1.4
- Presentations Addon: 0.1.0
